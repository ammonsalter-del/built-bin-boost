# Changelog

All notable changes to Build, Bin, Boost: The R&D Portfolio Simulation will be documented in this file.

## [1.1] - 2025-01-XX

### Changed
- Extended legacy project timelines to ensure active portfolio through Year 2
  - SmartLock 2.0: 6→8 quarters
  - Regional Language App: 5→8 quarters
  - Executive Analytics Dashboard: 7→9 quarters
  - OEM White-Label Program: 7→9 quarters
  - Cost-Reduced Sensor v2: 6→8 quarters
  - Cloud Migration Initiative: 7→9 quarters
- Extended Year 1 new project minimum timelines from 7 to 8 quarters across all archetypes
- Fixed progress display to cap at 99% until projects reach actual completion quarter

### Fixed
- Resolved issue where projects could show 100% progress while still active and RED status
- Prevented empty portfolio scenarios when players actively kill underperforming projects

## [1.0] - 2025-01-XX

### Added
- Initial release of Build, Bin, Boost simulation
- Complete 8-quarter (2-year) gameplay spanning project selection through outcomes
- Legacy portfolio review phase with 7 inherited projects
- Process design phase with 4 decision-making models
- Committee selection with 15+ stakeholder archetypes
- Evaluation approach selection (scoring matrix, intuitive, staged gates)
- Project assessment with multi-dimensional scoring
- Committee feedback and comparison views
- Portfolio builder with matrix visualisations
- 60+ management dilemmas across 6 categories
- Corporate events system with market shifts and crises
- Year 2 budget review and new project opportunities
- Career outcome system based on portfolio performance
- Detailed project templates with milestones, TRL, IP analysis
- Strategic bucket system for portfolio balance
- Kill quality scoring to evaluate termination decisions

### Features
- Self-contained single HTML file (26,000+ lines)
- No external dependencies
- Procedurally generated project pools for replayability
- Responsive design for desktop and tablet
- Sound effects and visual feedback
- Comprehensive results analysis and scoring
