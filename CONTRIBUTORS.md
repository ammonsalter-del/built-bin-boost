## How to Contribute

We welcome contributions to improve this educational simulation:

1. **Report Issues**: Open a GitHub issue for bugs or problems
2. **Suggest Enhancements**: Propose new features, dilemmas, or project archetypes
3. **Submit Pull Requests**: Fork the repository and submit improvements
4. **Share Feedback**: Let us know how you've used the simulation in teaching

### Areas for Contribution

- New management dilemma scenarios
- Additional project archetypes
- Localisation for different regional contexts
- Accessibility improvements
- Mobile responsiveness enhancements
- Assessment and analytics features
